# Reaction Storm

### Description

À tout moment, le bot peut réagir à un message sur vos serveurs. Les premiers à cliquer remporteront des RP ! Alors, soyez à l'affût !

### Récompenses

* Pour les **3 premiers** à réagir : **+20 RP** chacun.
